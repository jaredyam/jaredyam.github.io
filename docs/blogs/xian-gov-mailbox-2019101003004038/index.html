2019年10月10日 - 阎良籍异地出租车常年在西安地铁口拼座揽客扰乱地铁口正常秩序，无人管理

市民举报

阎良籍出租车，常年在地铁三号线香湖湾地铁口揽客，拼座，起初个别出租车停靠揽客，无人管理，到后来成群结队早晚高峰两排停车，司机一拥而上，围在地铁出口，当有乘客出地铁就围上去，阎良25元差一位，差一位，从早到晚不间断，严重扰乱地铁口秩序，违停造成东三环辅道堵车严重，驾驶员素质地下，随地吐痰，乱扔烟头。引起地铁口环卫工人，逛世园会游客等极为不难，希望管理部门从源头根治这一乱象，达到治标治本的效果，还广大市民一个良好的环境。谢谢！

政府回复

您好： 您通过西安12345市民热线服务平台微信提交的诉求（编号为2019101003004038），已收悉。现回复如下： 针对您反映情况，阎良区交通运输局召开专题工作会，要求运输企业加强驾驶员的规范营运、文明驾驶、安全出行教育，并提高个人素质，禁止不文明行为，同时要求企业到香湖湾地铁口进行检查劝返，同时组织稽查人员和公司人员在高速路口、客运站门口前往西安的出租车进行检查，对拼座、议价不按规定运营的车辆进行处理；对香湖湾地铁口运政人员和市处稽查人员交办的违章阎良出租车进行批评教育并予以严管重罚；利用车辆GPS进行实时监控管理，发现滞留车辆立即通知公司及时召回。 下一步，阎良区交通运输局积极与市级部门加强联系，齐抓共管，共同治理，维护好出租行业市场营运秩序，为市民提供一个安全、文明的出行环境。 感谢您对我们工作的支持与理解。
